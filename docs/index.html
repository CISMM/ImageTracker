About
 

ImageTracker is an application for the analysis and measurement of motion in video. It was designed specifically for the analysis of fluorescent speckle microscopy of cellular mitosis. ImageTracker is written using open-source software libraries such as ITK and wxWidgets. See the About page for more information.
